- in the browser, login to npm, and create an organization and a team inside it
- lets say the name of the organization = `am-ogs`

- on the command line, run:
- `npm login` (once)
- `npm init --scope=<your_org_name>`
    - `npm init --scope=am-ogs`

- `npm publish`
